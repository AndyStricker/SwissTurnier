Swissturnier
============

Overview
--------

A very simple, cross platform swiss turnier software. 

Story
-----

For an amateur badminton turnier, we used to have round-robin style
turnier.Then we got bigger and couldn't keep this form and had to switch
to the swiss style turnier.
